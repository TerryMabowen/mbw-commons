计算规则

1.现金流利润 = 经营性利润 + 非经营性收入-投资支出-非经营性支出

2.经营利润 = 经营收入-经营支出

3.绩效成本 = 经营支出-绩效支出扣减项+绩效成本增加项

4.绩效利润 = 经营收入-绩效成本*绩效利润比例

5.分总部分摊利润= (（分总部）绩效利润 + （分公司）经营性收入*比例系数4 ) * 比例系数5

6.实际绩效利润=绩效利润 + 分总部分摊利润

7.核算成本 = 经营支出-核算成本扣减项+核算承恩增加项

8.核算利润=确认收入 + 利息收入 - 核算成本

9.分总部分摊核算利润= （（分总部）核算利润 + （分公司）确认收入 * 比例系数4）* 比例系数5

10.实际核算利润=核算利润 + 分总部分摊核算利润
